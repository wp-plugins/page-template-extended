== Description ==

The Page Templates Extended plugin - Makes it possible to create templates for a specific page by its ID - like categories. If the page hasn't a template assigned and its a subpage, the parent page template will be used if it exists.

== Installation ==

1. Upload `pagetemplates_ex.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the "Plugins" menu in WordPress
3. Create a page template in our theme (page-*ID*.php).
